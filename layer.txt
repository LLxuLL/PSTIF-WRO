================================================================================
DATASETS
================================================================================

1. Heart Disease (UCI)
   - Files: cleveland.data, long-beach-va.data, switzerland.data, hungarian.data
   - Size: ~303 samples
   - Features: 13 attributes
   - Task: Binary classification (heart disease diagnosis)
   - Partitions: 6 (demographic, symptoms, vital_signs, lab_tests, ecg, angiography)

2. Sepsis Survival (UCI)
   - Files: s41598-020-73558-3_sepsis_survival_*.csv
   - Size: ~110K samples
   - Features: 3 attributes
   - Task: Survival prediction
   - Partitions: 2 (demographic, clinical)

3. German Credit (UCI)
   - File: german.data
   - Size: 1,000 samples
   - Features: 20 attributes
   - Task: Credit risk assessment
   - Partitions: 7 (financial_status, credit_history, loan_details, employment, demographic, assets, other)

4. Credit Card Fraud
   - File: creditcard.csv
   - Size: 284,807 transactions
   - Features: 30 attributes (V1-V28, Time, Amount)
   - Task: Fraud detection (highly imbalanced)
   - Partitions: 5 (time, pca_components_1, pca_components_2, pca_components_3, amount)

5. Amazon Electronics (5-core)
   - File: Electronics.csv
   - Size: ~1.9M interactions
   - Features: 9 attributes
   - Task: Rating prediction/recommendation
   - Partitions: 3 (user_features, item_features, interaction)

6. NYC Taxi Yellow
   - File: yellowtaxi_data.csv
   - Size: ~3M trips
   - Features: 11 attributes
   - Task: Trip duration prediction
   - Partitions: 4 (temporal, pickup_location, dropoff_location, trip_info)

================================================================================
KEY THEORETICAL CONTRIBUTIONS
================================================================================

1. IF-Measure Space Differential Geometry
   - First rigorous embedding of intuitionistic fuzzy numbers into Wasserstein space
   - Proof that measure-based aggregation preserves convex combination closure

2. Differentiable Robust Optimization via OT
   - Equivalence between traditional quasi-concave robust optimization and
     Wasserstein ball measure projection
   - Convergence rate O(e^{-λ}) for Sinkhorn robust layer

3. Partition Graph Message Passing Upper Bound
   - PW-GCN expressiveness no less than Weisfeiler-Lehman graph isomorphism test
   - Graph-theoretic guarantee for attribute partitioning

================================================================================
USAGE EXAMPLES
================================================================================

Training:
    python train.py --dataset heart_disease --epochs 200 --batch_size 16

Evaluation:
    python evaluate.py --checkpoint outputs/checkpoints/best_model.pth \
                       --dataset heart_disease \
                       --adversarial_test

Custom Config:
    python train.py --config configs/custom.yaml
